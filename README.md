# 樂器行庫存與租借管理系統

使用 Java 實作木管樂器庫存、租借收費與自助式租還狀態管理，並提供命令列版本與 Java Swing GUI 視覺化版本。

## 系統功能

- 新增樂器資料，並存入 `ArrayList`
- 依樂器編號精確查詢資料
- 顯示所有庫存資料
- 輸入租借天數後，自動計算含保養費的總租金
- 租借樂器，將樂器狀態標記為「已被借走」
- 歸還樂器，將樂器狀態恢復為「在庫可租」
- 計算租金時會檢查租借狀態，已被借走的樂器不可重複計算新的租借費用
- 提供 Java Swing GUI 視覺化操作介面

## 物件導向設計

- `Woodwind`：父類別，封裝編號、品牌、日租金、租借狀態
- `Flute`：長笛，專屬屬性為是否開孔，保養費 150 元
- `Clarinet`：豎笛，專屬屬性為豎笛種類，保養費 200 元
- `Saxophone`：薩克斯風，專屬屬性為薩克種類，保養費 250 元
- `Oboe`：雙簧管，無專屬擴充屬性，保養費 300 元
- `Bassoon`：低音管，無專屬擴充屬性，保養費 350 元
- `InventoryManager`：使用 `ArrayList<Woodwind>` 管理資料
- `Main`：選單式主程式
- `InstrumentRentalGUI`：Java Swing 視覺化主視窗

## 租借狀態管理

每一筆樂器資料都有 `isRented` 狀態欄位：

- `false`：在庫可租
- `true`：已被借走

查詢資料與顯示所有庫存時，系統會一起顯示目前租借狀態。

## 命令列主選單

```text
1. 新增樂器資料
2. 依編號查詢樂器
3. 顯示所有庫存
4. 計算租借總金額
5. 租借樂器
6. 歸還樂器
0. 離開系統
```

## Swing GUI 介面

GUI 主視窗標題為「樂器行庫存與租借管理系統」，視窗大小約為 900 x 600。

介面包含：

- 上方標題區：顯示系統名稱
- 左側功能按鈕區：垂直排列所有功能按鈕
- 右側資料顯示區：使用 `JTextArea` 顯示樂器資料、查詢結果、租金結果與租借狀態

GUI 按鈕功能：

- 新增樂器
- 查詢樂器
- 顯示所有樂器
- 計算租金
- 租借樂器
- 歸還樂器
- 清除畫面
- 離開系統

- `B001` Fox 低音管

## 計費規則

- 長笛：日租金 × 天數 + 150 元
- 豎笛：日租金 × 天數 + 200 元
- 薩克斯風：日租金 × 天數 + 250 元
- 雙簧管：日租金 × 天數 + 300 元
- 低音管：日租金 × 天數 + 350 元
